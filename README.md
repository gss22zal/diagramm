# 🏥 Анализатор медицинских отчётов и визуализатор

Настольное приложение на Python для парсинга PDF-отчётов о медицинских тестах, выгружаемое из приложения "Квантовый Магнитно-резонансный Биоанализатор", хранения структурированных данных в базе данных SQL Server и интерактивного просмотра с визуализацией параметров здоровья в виде диаграмм.

## ✨ Возможности

- 📄 **Парсинг PDF**: Извлечение информации о пациенте, датах тестирования, системах организма, параметрах и значениях из стандартизированных медицинских отчётов
- 🗄️ **Интеграция с БД**: Сохранение распарсенных данных в SQL Server (`health_db`) с полной реляционной структурой
- 📊 **Система референсных значений**: Хранение нормативных диапазонов и уровней отклонений (`+`, `++`, `+++`) для автоматической интерпретации результатов
- 🖥️ **Интерактивный GUI (Tkinter)**:
  - Каскадные фильтры: `Пациент → Дата → Система организма`
  - Цветовая кодировка результатов (зелёная/жёлтая/синяя/красная зоны)
  - Обновление данных одним кликом и удаление записей
- 📈 **Динамические диаграммы**: Графики на основе Matplotlib, показывающие историю параметров с визуальными референсными зонами
- 📦 **Пакетный импорт**: Обработка всех PDF в целевой папке через `save_health_data.py`
- 🔐 **Безопасная конфигурация**: Учётные данные БД и пути хранятся в `.env`

## 📥 Установка

1. **Клонируйте или скачайте** файлы проекта
2. **Установите зависимости**:
   ```bash
   pip install -r requirements.txt
   ```
3. **Убедитесь, что SQL Server и ODBC Driver** установлены и доступны

## ⚙️ Конфигурация

Создайте файл `.env` в корне проекта:

# Подключение к базе данных
DB_SERVER=localhost
DB_NAME=health_db
DB_USER=ваш_пользователь
DB_PASSWORD=ваш_пароль
DB_DRIVER={ODBC Driver for SQL Server}

# Папка для импорта PDF (относительно расположения скрипта)
PDF_FOLDER=pdf_files

## 🗃️ Настройка базы данных
Для создания таблицы в базе данных и заполнения референсных значений выполните:
```bash
python init_db.py
```

## 🚀 Использование

### 1. Импорт PDF-отчётов
Поместите все медицинские PDF-отчёты в папку, указанную в `.env` (`PDF_FOLDER`), затем выполните:
```bash
python save_health_data.py
```
Скрипт распарсит все файлы, извлечёт данные и вставит их в `health_results`.

### 2. Запуск GUI приложения
```bash
python main.py
```

#### Рабочий процесс в GUI:
1. Выберите **Пациента** → список дат автоматически обновится
2. Выберите **Дату тестирования** → список систем обновится
3. Выберите **Систему организма** → таблица загрузится с цветовой кодировкой результатов
4. Нажмите **📊 Диаграммы** чтобы открыть окно графиков. Выберите любой параметр для просмотра его истории с референсными зонами
5. Используйте **🗑 Удалить** для удаления всех записей выбранного пациента за выбранную дату
6. Используйте **📥 Импорт PDF** для запуска пакетного импорта прямо из GUI
